RESUMEN EJECUTIVO
─────────────────
Este documento mapea las prioridades de desarrollo para convertir TradeIQ de un
"generador de señales" a una "plataforma de aprendizaje de trading" con capacidad
de validación y auditoría de estrategias.

Stack actual: Next.js 16 + Prisma + SQLite + Polygon.io + Alpaca Markets


═══════════════════════════════════════════════════════════════════════════════
FASE 1: CIMIENTOS (Semanas 1-4)
═══════════════════════════════════════════════════════════════════════════════

OBJETIVO: Cerrar el ciclo completo señal → ejecución → resultado

El problema actual: El usuario recibe una señal LONG/SHORT con SL y TP pero no
hay forma de saber si realmente ejecutó la operación, si fue correcta o qué pasó
después. La Bitácora existe pero está desconectada del motor de análisis.

│ últimos 6 meses usando datos históricos de Polygon.io.                    │
│                                                                             │
│ IMPLEMENTACIÓN:                                                             │
│                                                                             │
│ 1. CREAR MICRO-SERVICIO DE BACKTESTING                                    │
│    Archivo: mini-services/backtest-engine.ts                              │
│    Nota: Ya existe carpeta mini-services en el repo                       │
│                                                                             │
│    Funcionalidad:                                                          │
│    - Recibe: symbol, timeframe, startDate, endDate, vectorConfig          │
│    - Obtiene datos históricos de Polygon.io                               │
│    - Corre el motor de confluencia en cada candle                         │
│    - Registra todas las señales que se hubieran generado                  │
│    - Calcula P&L de cada señal (asumiendo ejecución)                      │
│    - Retorna: total trades, wins, losses, win rate, max drawdown          │
│                                                                             │
│    Pseudocódigo:                                                           │
│    ```                                                                     │
│    function backtestStrategy(                                             │
│      symbol: string,                                                      │
│      vectorConfig: VectorConfig,                                          │
│      startDate: Date,                                                     │
│      endDate: Date,                                                       │
│      timeframe: "1min" | "5min" | "15min" | "1hour" | "1day"             │
│    ) {                                                                    │
│      // 1. Obtener candles históricos                                     │
│      const candles = await polygon.getCandles(symbol, startDate, endDate)│
│                                                                             │
│      // 2. Iterar cada candle                                             │
│      const signals = []                                                   │
│      for (let i = 50; i < candles.length; i++) {                         │
│        // Construir OHLCV array hasta este punto                          │
│        const historicalData = candles.slice(0, i + 1)                    │
│                                                                             │
│        // 3. Correr análisis                                              │
│        const analysis = runConfluenceEngine(                              │
│          historicalData,                                                  │
│          vectorConfig                                                     │
│        )                                                                  │
│                                                                             │
│        // 4. Si hay señal, registrarla                                    │
│        if (analysis.signal && analysis.confluenceScore > threshold) {     │
│          signals.push({                                                   │
│            timestamp: candles[i].timestamp,                               │
│            direction: analysis.direction,                                 │
│            entryPrice: candles[i].close,                                  │
│            // ... resto de datos                                          │
│          })                                                               │
│        }                                                                  │
│      }                                                                    │
│                                                                             │
│      // 5. Calcular resultados                                            │
│      const results = calculateBacktestMetrics(signals, candles)          │
│      return results                                                       │
│    }                                                                      │
│    ```                                                                     │
│                                                                             │

├─────────────────────────────────────────────────────────────────────────────┤

│                                                                             │
│    Crear función que:                                                      │
│    - Obtiene órdenes cerradas de Alpaca                                  │
│    - Las sincroniza con la BD local                                       │
│    - Calcula P&L real                                                     │
│    - Ejecutar cada 5 minutos                                              │
│                                                                             │
│    ```                                                                     │
│    import Alpaca from '@alpacahq/alpaca-trade-api'                       │
│                                                                             │
│    async function syncTrades() {                                          │
│      const alpaca = new Alpaca({                                          │
│        apiKey: process.env.ALPACA_API_KEY,                                │
│        secret: process.env.ALPACA_API_SECRET                              │
│      })                                                                    │
│                                                                             │
│      // Obtener órdenes cerradas                                         │
│      const closedOrders = await alpaca.getOrders({                       │
│        status: 'closed',                                                  │
│        limit: 100,                                                        │
│        nested: true                                                       │
│      })                                                                    │
│                                                                             │
│      for (const order of closedOrders) {                                  │
│        // Buscar si ya existe en nuestra BD                              │
│        const existing = await db.trade.findUnique({                      │
│          where: { alpacaOrderId: order.id }                              │
│        })                                                                 │
│                                                                             │
│        if (existing) continue // Ya fue procesado                         │
│                                                                             │
│        // Calcular P&L                                                    │
│        const entryPrice = order.filled_avg_price                         │
│        const exitPrice = order.filled_avg_price // si fue cerrada        │
│        const quantity = order.filled_qty                                  │
│        const pnl = (exitPrice - entryPrice) * quantity                   │
│        const pnlPercent = ((exitPrice - entryPrice) / entryPrice) * 100  │
│                                                                             │
│        // Crear en BD                                                    │
│        await db.trade.create({                                           │
│          data: {                                                         │
│            symbol: order.symbol,                                         │
│            direction: order.side === 'buy' ? 'LONG' : 'SHORT',          │
│            entryPrice,                                                   │
│            exitPrice,                                                    │
│            quantity,                                                     │
│            pnl,                                                          │
│            pnlPercent,                                                   │
│            alpacaOrderId: order.id,                                      │
│            status: 'CLOSED',                                             │
│            closedAt: new Date(order.filled_at)                           │
│          }                                                               │
│        })                                                               │
│      }                                                                   │
│    }                                                                      │
│                                                                             │
│    // Ejecutar cada 5 minutos                                            │
